# Yuranny Cabral-Calderin — academic website

This is a simple static website for GitHub Pages. It does not require a build step, a database, cookies, analytics, or third-party scripts.

## Upload to GitHub

1. Open the repository used for `ycabralcalderin.github.io`.
2. Replace the existing `index.html` and `style.css` with the two files in this folder.
3. Commit the changes to the repository's default branch.
4. In **Settings → Pages**, confirm that GitHub Pages is publishing from the root of that branch.

The update can take a few minutes to appear.

## Optional portrait

The supplied version uses a typographic `YC` panel, so no image file is required. If a portrait is added later, use a compressed `.webp` or `.jpg` file and include descriptive alternative text.

## Privacy

The public site intentionally excludes the private address, telephone number, date of birth, citizenship, and family information contained in the full CV. It uses only professional contact details and public academic-profile links.

Before naming students or early-career researchers on the site, confirm their preferred name, current role, project description, and consent to be listed.
